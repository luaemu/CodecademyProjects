Common CSS Properties

Property name Default value Possible values Description
color #000/black green, #A1B2C3, rgb(0, 153, 255), rgba(255, 0, 255, 0.75) Sets the foreground color of an element's text.
background-color #fff/white green, #A1B2C3, rgb(0, 153, 255), rgba(255, 0, 255, 0.75) Sets the background color of an element.
margin 0 (on all sides) 12px, 36pt, 10%, auto Controls how much space is placed around the outer sides of an element. Shorthand for the properties margin-top, margin-left, etc.
margin-top 0 12px, 36pt, 10%, auto Same as above, but specifies the margin to be placed specifically on the top side of the element.
